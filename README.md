### Требования

1. Валидации должны выполняться с использованием утверждений
2. Page object паттерн используется для писания страниц и элементов
3. Используйте корпоративную среду веб-автоматизации

## Сценарии

Сайт: https://userinyerface.com/

### Test case 1.

| Шаги                                                                                                                  | Ожидаемые результаты         | 
|-----------------------------------------------------------------------------------------------------------------------|------------------------------|
| 1. Перейти на домашнюю страницу.                                                                                      | Открылась домашняя страница. | 
| 2. Кликнуть по ссылке (в тексте "Please click HERE to GO to the next page") для перехода на следующую страницу.       | Открылась карточка "1".      | 
| 3. Введите случайные действующий пароль, адрес электронной почты, примите условия пользования и нажмите кнопку "Next" | Открылась карточка "2".      | 
| 4. Выберите 2 случайных интереса, загрузите изображение, нажмите кнопку "Next"                                        | Открылась карточка "3".      |
